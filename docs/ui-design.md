# UI 设计文档

## 低保真 → 高保真设计流程

### 工具链

1. **ASCII 线框图**（docs/wireframes.md）— 低保真原型
2. **Figma / Sketch** — 高保真视觉设计
3. **Google Speech-to-Text** — 语音指令生成页面框架
4. **Tailwind CSS** — 直接编码实现高保真界面

### 实际实现方式

本项目采用 **设计即代码** 的快速迭代方式：

1. ASCII 线框图确定页面布局
2. Tailwind CSS 在代码中直接实现样式
3. 浏览器实时预览（`npm run dev` 热重载）
4. 迭代调整直到满意

## 设计规范

### 配色方案

| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色调 | 森林绿 | #7E9D82 |
| 主色调深 | 深森林绿 | #6F8B73 |
| 暗色模式背景 | 深绿黑 | #1e2a1e |
| 危险/删除 | 红色 | #ef4444 |
| 文字主色 | 深灰 | #334155 |
| 文字辅色 | 浅灰 | #94a3b8 |

### 组件风格

| 元素 | 样式 |
|------|------|
| 弹窗 | 圆角 12px、毛玻璃背景、阴影 |
| 按钮 | 圆角 6px、过渡动画 |
| 输入框 | 圆角 6px、focus 绿色光环 |
| 头像 | 圆形、边框、阴影 |
| 标记 | 圆形 + 白色边框 + 阴影 |

### 响应式断点

| 断点 | 宽度 | 适配 |
|------|------|------|
| 手机 | < 768px | 全屏布局、右缩搜索栏 |
| 桌面 | ≥ 768px | 搜索栏左置 360px、面板固定 |

### 特殊适配

- iPhone 刘海屏：`viewport-fit: cover` + `env(safe-area-inset-*)`
- 动态视口：`100dvh` + `-webkit-fill-available` 回退
- 暗色模式：CSS `filter: invert() hue-rotate()` 反转地图瓦片
